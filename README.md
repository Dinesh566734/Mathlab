# Campus Connect

**Engage, Collaborate, Grow**

Campus Connect is a digital ecosystem designed to bridge the gap between students, faculty, and alumni. It provides a platform for seamless communication, discovering opportunities, and accessing academic resources.

## Features

-   **Seamless Communication**: Structured channels and instant announcements to connect with peers and mentors.
-   **Opportunities Hub**: Access to exclusive internships, campus events, and competitions.
-   **Smart Resources**: A library of student-powered notes and academic materials.
-   **Math-LAB**: A specialized module for Class 11+ mathematics resources.

## Math-LAB

Math-LAB is a key component of Campus Connect, focusing on mathematics education. It leverages the power of community to provide high-quality resources.

### Key Features
-   **Topic-based Learning**: Resources are organized by topics and categories (e.g., Algebra, Calculus).
-   **Contributor Network**: Content is curated by students and verified contributors.
-   **Dual-mode Learning**: Each resource page offers side-by-side access to:
    -   **Study Notes (PDF)**: High-quality notes embedded directly from Google Drive.
    -   **Video Explanations**: Curated YouTube videos for in-depth understanding.

## Tech Stack

-   **Frontend**: HTML5, CSS3, JavaScript (Vanilla + ES6 Modules).
-   **Backend / Database**: Firebase Firestore (for dynamic content management in Math-LAB).
-   **Styling**: Custom CSS with responsive design and Dark Mode support.

## Setup & Usage

Since this is a client-side application, no complex build process is required.

1.  **Clone the repository**:
    ```bash
    git clone <repository-url>
    ```
2.  **Open the application**:
    -   Simply open `index.html` in your web browser to access the main Campus Connect portal.
    -   Navigate to the "Math-LAB" section using the slider or open `mathlab/index.html` directly.
